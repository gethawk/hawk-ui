## Installation
`$ npm install @hawk-ui/progress-bar --save`


## Usage


#### Progress Bar With Vary Color
[Demo](https://hawk.wallnit.com/#!/ProgressBar/1)
```js
<ProgressBar
  value={7}
  maxRange={10}
/>
```


#### Progress Bar With Same Color
[Demo]()
```js
<ProgressBar
  value={85}
  maxRange={100}
  isColorVary={false}
  isProgress
/>
```
