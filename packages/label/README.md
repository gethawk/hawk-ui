## Installation
`$ npm install @hawk-ui/label --save`


## Usage


#### Without Required
[Demo]()
```js
<Label
  title="Label"
/>
```


#### With Required
[Demo]()
```js
<Label
  title="Label"
  isRequired
/>
```


#### With Left icon
[Demo]()
```js
<Label
  icon="fa fa-copy"
  title="Label"
/>
```


#### With Right icon
[Demo]()
```js
<Label
  icon="fa fa-copy"
  iconPlacement="right"
  title="Label"
  isRequired
/>
```
