## Installation
`$ npm install @hawk-ui/image --save`


## Usage


#### Image
[Demo]()
```js
<div style={{ width: '200px' }}>
  <Image
    src={null}
    fallbackSrc="https://www.google.com/images/branding/googlelogo/2x/googlelogo_color_272x92dp.png"

</div>
```
