#### Basic Image Usage:

```js
<div className="styleguidist__btns-wrap">
  <br /><br />
  <div style={{ width: '200px' }}>
    <Image
      src="https://www.google.com/images/branding/googlelogo/2x/googlelogo_color_272x92dp.pn"

      role="show image"
      title="google"
    />
  </div>
</div>
```
